# Bookstore Demo Extra

Fork of the [Bookstore Demo](../bookstore/README.md) to demonstrate the use of extra features of Remix, including [typed form data](../../packages/form-data-typed-middleware/README.md), [dependency injection](../../packages/router-services-middleware/README.md) and using the [fetch-router-extra](../../packages/fetch-router-extra/README.md) utilities.

See the [Bookstore Demo](../bookstore/README.md) for more information.

## Running the Demo

```bash
cd demos/bookstore-extra
pnpm install
pnpm start
```

Then visit http://localhost:44100
